NIXL KVBench
============

This workload (`test_template_name` is ``NIXLKVBench``) runs NIXL KV-cache benchmarking for key-value store performance testing.

Usage Example
-------------

Test TOML example:

.. code-block:: toml

   name = "my_nixl_kvbench_test"
   description = "Example NIXL KVBench test"
   test_template_name = "NIXLKVBench"

   [cmd_args]
   docker_image_url = "..."
   model = "./examples/model_deepseek_r1.yaml"
   model_config = "./examples/block-tp1-pp16.yaml"
   backend = "POSIX"
   num_requests = 1
   source = "file"
   num_iter = 16
   page_size = 256
   filepath = "/data"

Test Scenario example:

.. code-block:: toml

   name = "nixl-kvbench-test"

   [[Tests]]
   id = "kvbench.1"
   num_nodes = 1
   time_limit = "00:10:00"

   test_name = "my_nixl_kvbench_test"

Test-in-Scenario example:

.. code-block:: toml

   name = "nixl-kvbench-test"

   [[Tests]]
   id = "kvbench.1"
   num_nodes = 1
   time_limit = "00:10:00"

   name = "my_nixl_kvbench_test"
   description = "Example NIXL KVBench test"
   test_template_name = "NIXLKVBench"

     [Tests.cmd_args]
     docker_image_url = "..."
     backend = "UCX"
     source = "memory"
     op_type = "READ"
